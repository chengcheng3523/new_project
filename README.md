## 需求
1.藥、肥、其他，
  單位使用量要計算，改不可輸入。
2.資材都要套入，出入庫的資材名稱。
3.面積加單位，(公頃、公畝、幾甲)並加上換算
4.栽培工作(作物)選單，僅可選擇，種苗登記裡的作物
5.農地編號，僅可選擇農地資訊，可以只顯示ID
6.養液不要
7.有害生物防治=藥
以此類推
